# Web App Icons

This folder contains SMART ReplySuite icon files for browser shortcuts, favicons and web app metadata.

- `replysuite-icon.png`: master icon.
- `replysuite-icon-512.png`, `replysuite-icon-256.png`, `replysuite-icon-128.png`: browser and manifest icons.
- `replysuite-icon.ico`: optional favicon format.

SMART ReplySuite is a browser-based app.
